# Technical Finding / Milestone

**Context**: CALAIRE-APP / Etapa 3 del portal de rondas EA

## The Discovery

La Etapa 3 quedó operativa en el árbol actual de `/home/w182/w421/calaire-app` con el flujo completo de gestión de participantes por ronda:

- ruta protegida `/dashboard/rondas/[id]/participantes`
- búsqueda de usuarios WorkOS por correo
- asignación de participantes a la ronda
- listado con estado de envío (`pendiente` o con conteo de envíos)
- eliminación de participantes mientras la ronda no esté cerrada

La validación de compilación con `npm run build` terminó limpia en 2026-04-21, confirmando que la ruta dinámica y sus server actions están integradas correctamente en Next.js 16.

También quedó confirmado que el esquema de Supabase necesario para esta etapa incluye `ronda_participantes.email`, por lo que el usuario debe volver a ejecutar `db/schema.sql` en el proyecto Supabase activo antes de probar el flujo end-to-end.
