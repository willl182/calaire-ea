# Technical Finding / Milestone

**Context**: CALAIRE-EA Logseq graph — planificacion documental de la prueba piloto

## The Discovery

Se restauro el contexto del proyecto con `continue` y se identifico que la necesidad actual no era continuar el desarrollo del portal `calaire-app`, sino planificar un informe operativo dentro del grafo `calaire-ea`.

El plan creado consolida una estructura auditable para documentar la ejecucion de las dos rondas piloto activas del esquema [[CALAIRE-EA]], desde la invitacion de participantes hasta el cierre operativo y estadistico.

## Key Files Affected

- `logs/plans/260506_2143_plan_informe-operativo-pruebas-piloto.md`
- `logs/CURRENT_SESSION.md`

## Key Decisions Preserved

- El informe debe cubrir dos rondas:
  - [[Ronda 7]]: CO y SO2 con [[SIATA]].
  - [[Ronda 8]]: O3, NO y NO2 con [[SIATA]] y [[Universidad Pontificia Bolivariana]].
- El informe debe separar resultados operativos de resultados estadisticos.
- El informe debe declarar el caracter formativo del piloto por n bajo.
- La coherencia de `sigma_pt` y el hito T-3 quedan como controles criticos antes de emitir version final.
