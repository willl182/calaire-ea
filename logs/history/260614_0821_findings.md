# Technical Finding / Milestone

**Context**: Publicacion HTML de fichas resumen SGC PEA

## The Discovery

Las fichas resumen ya tenian estructura Markdown suficientemente consistente para publicarse como HTML sin reescritura de contenido. La mejor UX para el mapa no requiere modificar las fichas fuente, sino generar una capa HTML derivada con plantilla comun, retorno al mapa, tabla de contenido, tablas estilizadas y acceso secundario al Markdown.

El mapa solo contiene 47 nodos, mientras que existen 58 fichas `ficha_*.md`. Por eso se creo un indice HTML adicional para cubrir las fichas sin nodo directo.

## Key Files Affected

- `docs/documentacion_sgc/mapa_navegacion_sgc_pea.html`
- `docs/documentacion_sgc/fichas_resumen/build_fichas_html.sh`
- `docs/documentacion_sgc/fichas_resumen/ficha_resumen.css`
- `docs/documentacion_sgc/fichas_resumen/ficha_resumen_template.html`
- `docs/documentacion_sgc/fichas_resumen/index.html`
- `docs/documentacion_sgc/fichas_resumen/ficha_*.html`

## Validation

- Se confirmo paridad: 58 archivos `ficha_*.md` y 58 archivos `ficha_*.html`.
- El mapa ya no contiene enlaces generados hacia `ficha_${code}.md`.
- Validacion HTTP con Chromium headless confirmo carga `200` de mapa, indice, ficha y CSS.
- No se genero screenshot por cuota de disco en `/tmp`.
