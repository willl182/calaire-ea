## Resumen

Completada extracción de contenido modelo de 4 proveedores internacionales (JRC-ERLAP, UBA, Brno, UCLSB) para servir como referencia técnica en redacción de mejoras de la Comunicación Detallada del Ensayo de Aptitud.

## Logros

1. **JRC-ERLAP Safety (11.5)**: Extraída sección completa de seguridad industrial con elementos clave para adaptación a contexto colombiano
2. **UBA Quejas/Apelaciones**: Extraído modelo completo de manejo de quejas, alineado con ISO/IEC 17043:2023
3. **UBA σpt**: Extraída definición conceptual (reproducibility standard deviation) con nota sobre naturaleza parcial
4. **Brno Horn Procedure**: Extraída descripción de método robusto para muestras pequeñas (n < 8)
5. **UCLSB Subcontratación**: Extraído modelo de declaración explícita de subcontratación con responsabilidad del PT provider

## Hallazgos Técnicos Importantes

### 1. Adaptación Normativa
- JRC-ERLAP refiere normativa italiana (D.lgs. 81/2008)
- CALAIRE-EA debe adaptar a normativa colombiana de seguridad industrial (MinTrabajo)

### 2. σpt - Complejidad Detectada
- La definición de σpt como "reproducibility standard deviation" es conceptual
- Para Prueba Piloto (primera ronda), σpt no tiene datos históricos
- Requiere complemento con ISO 13528:2022 para definición formal aplicable a gases

### 3. Horn Procedure - Pertinencia
- Método relevante para rondas con n < 8
- Aplicable a gases contaminantes cuando participación es limitada
- Requiere criterio de activación documentado en CALAIRE-APP

### 4. Subcontratación - Opcionalidad
- Solo aplica si CALAIRE-EA subcontrata actividades
- Posibles aplicaciones: mezclas de gases, homogeneidad, transporte especializado
