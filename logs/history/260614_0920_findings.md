# Technical Finding / Milestone

**Context**: Mejora UX de fichas HTML SGC PEA

## The Discovery

Las fichas HTML derivadas pueden exponer metadatos operativos en el encabezado sin modificar los Markdown fuente. La plantilla `ficha_resumen_template.html` acepta metadatos Pandoc generados desde las tablas de identificacion, lo que permite mostrar tipo documental, estado, prioridad y clase de ficha como badges visibles.

Esto mejora la revision documental porque evita abrir la tabla de identificacion para distinguir fichas activas, preliminares, diferidas o no activas.

## Key Files Affected

- `docs/documentacion_sgc/fichas_resumen/build_fichas_html.sh`
- `docs/documentacion_sgc/fichas_resumen/ficha_resumen_template.html`
- `docs/documentacion_sgc/fichas_resumen/ficha_resumen.css`
- `docs/documentacion_sgc/fichas_resumen/ficha_*.html`
- `logs/CURRENT_SESSION.md`

## Validation

- Regeneracion ejecutada con `docs/documentacion_sgc/fichas_resumen/build_fichas_html.sh`.
- Paridad actual confirmada: 51 Markdown fuente y 51 fichas HTML generadas, excluyendo `ficha_resumen_template.html` del conteo.
- `index.html` contiene 51 enlaces de ficha.
- Todas las fichas HTML generadas contienen `hero-meta` y badges de tipo, estado, prioridad y clase.
- Todas las fichas mantienen referencia a `ficha_resumen.css` y `ficha_names.js`.
- El mapa no contiene enlaces crudos a fichas Markdown.
