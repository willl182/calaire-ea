# Technical Finding - Timeline PNG desde SVG con Inkscape

**Context**: Renderizado de timeline Mermaid con colores por columna

## El Hallazgo

El SVG generado por `mmdc` respeta correctamente los colores del timeline con `theme: "base"` y `cScale`. Sin embargo, el PNG rasterizado por `mmdc` no siempre preserva el mapeo de colores. La conversión desde SVG a PNG con Inkscape mantiene los colores esperados.

## Solución Aplicada

1. Generar SVG con `mmdc`:
   ```bash
   mmdc -i docs/timeline.md -o docs/informes/timeline_piloto.svg -w 1632 -H 576
   ```

2. Convertir SVG a PNG con Inkscape:
   ```bash
   inkscape docs/informes/timeline_piloto.svg \
     --export-filename=docs/informes/timeline_piloto.png
   ```

## Archivos Afectados

- `docs/timeline.md`
- `docs/informes/timeline_piloto.svg`
- `docs/informes/timeline_piloto.png`

## Aprendizaje

Cuando el PNG de Mermaid no respeta colores de timeline con `theme: base`, usar SVG como fuente y convertirlo con Inkscape para obtener un PNG fiel.
