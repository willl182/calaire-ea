# Technical Finding / Milestone

**Context**: Recodificación de guía del participante al SGC CALAIRE-EA
**Timestamp**: 2026-05-22 14:17

## Milestone

Se completó la recodificación del archivo `guia-participante-cargue-datos.html` de la codificación antigua `GL-PT-002` a la convención vigente del Sistema de Gestión de Calidad (SGC) del proyecto CALAIRE-EA.

## The Discovery

La guía del participante no encajaba como `DG-PSEA-02` (Documento General) porque su naturaleza es operativa: describe **cómo** el laboratorio participante usa CALAIRE-APP para reportar resultados. Tras revisar el Diccionario de Documentos SGC y el Inventario Documental, se determinó que el código correcto es **`I-PSEA-10`** → *Instructivo de Instrucciones a Participantes*.

**Corrección importante**: Inicialmente se consideró `I-PSEA-09`, pero el Inventario Documental del SGC ya asigna `I-PSEA-09` a *"Instructivo de Valor Asignado"*. La guía del participante corresponde a `I-PSEA-10`.

El documento ya existía en el diccionario/inventario como pendiente (TBD); este HTML es su primera materialización.

## Key Decisions

| Decisión | Resultado |
|----------|-----------|
| Tipo de documento | **Instructivo (I)**, no Documento General (DG) |
| Código asignado | **`I-PSEA-10`** |
| Versión | `v0.1` (notación del SGC) |
| Documento padre | `P-PSEA-20` (Procedimiento de Comunicación PT) |
| Formatos vinculados | `F-PSEA-05A` (Hoja de Registro), `F-PSEA-04` (Informe Resultados) |
| Normas aplicables | ISO/IEC 17043:2023, ISO 13528:2022 |
