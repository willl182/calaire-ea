# Problem Record (Troubleshooting)

**Severity**: Medium

## The Problem

El push por SSH falló inicialmente con:

- `ssh_askpass: exec(/usr/lib/ssh/ssh-askpass): No such file or directory`
- `Host key verification failed`

El entorno no tenía llaves SSH configuradas ni `known_hosts` para GitHub.

## The Solution / Workaround

1. Generar una llave SSH en el entorno.
2. Agregar `github.com` a `known_hosts`.
3. Registrar la llave pública en GitHub (Settings → SSH keys).
4. Reintentar `git push -u origin main`.
