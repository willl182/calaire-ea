# Technical Finding / Milestone

**Context**: CALAIRE-APP / Etapa 6 del portal de rondas EA

## The Discovery

La Etapa 6 quedó implementada técnicamente en `/home/w182/w421/calaire-app` sin romper la compilación de Next.js 16. El portal ya incorpora un endurecimiento suficiente para pruebas operativas:

- validación server-side del guardado de resultados por participante
- rechazo explícito de rondas no activas, usuarios no invitados, contaminantes/niveles inconsistentes y valores negativos
- validación reactiva en el formulario con errores visibles antes del envío
- páginas globales de `404` y acceso denegado
- `.env.example` y README alineados con setup real y despliegue en Vercel

La compilación `npm run build` terminó limpia el 2026-04-21 después de estos ajustes, por lo que el siguiente cuello de botella ya no es de código sino de configuración externa: variables de entorno, callback productivo de WorkOS y smoke test con credenciales reales.
