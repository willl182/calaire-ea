# Technical Finding / Milestone

**Context**: CALAIRE-EA Logseq graph — consolidación documental SGC y prueba piloto

## The Discovery

La documentación del SGC y de la prueba piloto estaba distribuida entre `docs/docs_sgc/` y `docs/prueba_piloto/`, con mezcla de documentos controlables, copias por ronda, borradores de IA, backups, informes antiguos y referencias externas.

Se consolidó la documentación en un único punto canónico:

- `docs/documentacion_sgc/01_bloque_general/` para documentación general del SGC.
- `docs/documentacion_sgc/02_prueba_piloto_rondas/` para documentación específica de rondas piloto.
- `docs/documentacion_sgc/z_archivos_para_descartar/` para documentos no aplicables, duplicados o de referencia histórica.

## Key Files Affected

- `docs/documentacion_sgc/README.md`
- `docs/documentacion_sgc/01_bloque_general/`
- `docs/documentacion_sgc/02_prueba_piloto_rondas/`
- `docs/documentacion_sgc/z_archivos_para_descartar/`
- `pages/Prueba Piloto.md`
- `pages/Inducción SST Persona Natural.md`
- `docs/calendario_ppiloto.md`
- `docs/descripcion_programa_ea.md`

## Key Decisions Preserved

- No se eliminaron archivos; los documentos no aplicables fueron movidos a descarte numerado.
- Las copias específicas de rondas quedaron separadas por tipo de ronda.
- Las plantillas y documentos marco quedaron bajo el bloque general.
- La carpeta `z_archivos_para_descartar/` debe revisarse antes de cualquier eliminación definitiva.
