## Perspectivas de Evaluación

| Perspectiva | Criterios |
|-------------|-----------|
| **Precisión Técnica** | Exactitud de datos, fechas, terminología ISO, consistencia interna, coherencia chronológica |
| **Claridad y Redacción** | Gramática, claridad, coherencia narrativa, tono profesional, concisión |
| **Completitud** | Cobertura del periodo, temas críticos abordados, omisiones, contexto suficiente |
| **Estructura y Formato** | Jerarquía de información, organización, tablas/figuras, consistencia, navegabilidad |
| **Accionabilidad** | Próximos pasos específicos, ownership claro, priorización, dependencias, seguimiento |

**Estados de Evaluación:**
- **OK:** Sin hallazgos significativos
- **Ajuste menor:** Mejoras opcionales o cosméticas
- **Ajuste mayor:** Correcciones necesarias antes de distribución
- **Crítico:** Errores que invalidan la información

## Instrucciones para el Modelo

1. Revisa cada una de las 6 secciones del informe ejecutivo
2. Para cada sección, evalúa las 5 perspectivas asignadas
3. Sé específico en los hallazgos, indicando ubicación exacta (tabla, párrafo, línea, subsección)
4. Proporciona correcciones concretas:
   - Para problemas de redacción: texto corregido
   - Para problemas técnicos: explicación técnica + corrección
   - Para problemas estructurales: sugerencia de reorganización
5. Completa los resúmenes finales (por sección y por perspectiva)
6. Identifica las Top 5 correcciones prioritarias (críticos y mayores)
7. Guarda tu revisión en: `docs/informes/260208_ie_01_rev#.md`
