# **Propuesta de Cronograma: Pruebas Piloto Ensayos de Aptitud \- Marzo - Mayo 2026**

## **1\. Introducción**

El presente documento detalla la propuesta logística y de calendario para la ejecución de los **Ensayos de Aptitud para la medición de gases contaminantes criterio** (SO₂, NO₂, CO y O₃).

Con el objetivo de maximizar la capacidad operativa y garantizar la calidad de los datos conforme a la norma **ISO/IEC 17043:2023**, se ha diseñado un esquema de **8 rondas (6 activas)** a realizarse durante el período de **marzo - mayo de 2026**. Las rondas previstas para febrero fueron canceladas por la preparación de las autoridades ambientales para las contingencias de marzo.

## **2\. Esquema Operativo de la Ronda (Ciclo Semanal)**

Para cada una de las cinco rondas, se mantendrá una estructura fija de **miércoles a sábado**, con retiro de equipos el día lunes. Esto permite estandarizar los tiempos de exposición y recolección de datos para todos los participantes.

### **Distribución de Actividades por Día**

| Día | Actividad Principal |
| :---- | :---- |
| **Miércoles** | **Recepción y Acondicionamiento** |
| **Jueves** | **Inicioy Ejecución de Ensayos (Día 1\)** |
| **Viernes** | **Ejecución de Ensayos (Día 2\)** |
| **Sábado** | **Ejecución y Cierre de Ensayos (Día 3\)** |
| **Lunes** | **Recolección de Equipos** |

## **5\. Requerimientos para Participantes**

Recordando las condiciones establecidas en la invitación piloto, cada participante asignado a una ronda deberá garantizar la presencia de su **analizador, calibrador dinámico, sistema de aire cero, generador de ozono y cilindro de gas patrón** el día miércoles a las **08:00 AM** para no retrasar el inicio del acondicionamiento.
