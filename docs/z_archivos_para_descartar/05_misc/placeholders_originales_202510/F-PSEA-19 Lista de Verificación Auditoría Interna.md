## Descripción

Checklist/lista de verificación para auditorías internas del SGC del PEA. Organizada por cláusula de ISO/IEC 17043:2023 (§4–8), incluye: requisito normativo, evidencia esperada, conformidad (C/NC/Obs), evidencia verificada y observaciones. Diseñada para cubrir todos los requisitos aplicables al esquema de EA de CALAIRE-EA. Soporte documental de P-PSEA-17.
