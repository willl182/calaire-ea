## Descripción

Matriz para identificar, evaluar y hacer seguimiento a los riesgos de imparcialidad del PEA. Incluye: amenaza identificada (interna/externa), tipo de amenaza (autorrevisión, familiaridad, intimidación, interés propio, defensa), probabilidad, impacto, nivel de riesgo, salvaguarda implementada, riesgo residual, responsable y estado. Crítica para CALAIRE-EA dado el riesgo "juez y parte" (laboratorio = PEA). Soporte documental de P-PSEA-19.
