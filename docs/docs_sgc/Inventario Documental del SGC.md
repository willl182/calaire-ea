## Objetivo

Este inventario proporciona una lista completa y clasificada de todos los documentos existentes en el Sistema de Gestión de Calidad (SGC) del proyecto CALAIRE-EA, identificando su tipo (DG/P/I/F), estado, y relación con los requisitos normativos de ISO/IEC 17025:2017, ISO/IEC 17043:2023 e ISO 13528:2022.

---

## Convenciones de Clasificación

| Prefijo | Tipo | Descripción | Ejemplo |
|---------|------|-------------|---------|
| DG | Documento General | Políticas, manuales, lineamientos marco | DG-PSEA-01 |
| P | Procedimiento | Define qué se hace, quién, cuándo, evidencia | P-PSEA-01 |
| I | Instructivo | Define cómo se ejecuta una actividad puntual | I-PSEA-01 |
| F | Formato | Registro/evidencia (Excel cuando aplique) | F-PSEA-01 |

| Capítulo | Requisitos Cubiertos | Requisitos Faltantes | Prioridad |
|----------|---------------------|---------------------|-----------|
| 4 (General Requirements) | Parcial | Confidencialidad, política de imparcialidad | Alta |
| 5 (Structural Requirements) | Parcial | Organización, estructura, autoridad | Media |
| 6 (Resource Requirements) | Parcial | Gestión de competencia, ambiente, proveedores externos | Alta |
| 7 (Process Requirements) | Parcial | Revisión de contratos, métodos, incertidumbre, registros | Alta |
| 8 (Management System Requirements) | Parcial | Control documental, registros, riesgos, mejora, NC, auditorías, revisión | Alta |

#### ISO/IEC 17043:2023

| Capítulo | Requisitos Cubiertos | Requisitos Faltantes | Prioridad |
|----------|---------------------|---------------------|-----------|

| 5 (Structural Requirements) | Parcial | Organización, estructura, autoridad | Media |

| 7 (Process Requirements) | Parcial | Planificación, diseño estadístico, valor asignado, H/E, reportes | Alta |
| 8 (Management System Requirements) | Parcial | Control documental, registros, riesgos, mejora, NC, auditorías, revisión, quejas, apelaciones | Alta |

| Capítulo | Requisitos Cubiertos | Requisitos Faltantes | Prioridad |
|----------|---------------------|---------------------|-----------|
