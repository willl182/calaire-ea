## Descripción

Instructivo para el control de calidad de los datos recibidos de los participantes y los datos generados internamente. Define: verificación de unidades, formato y cifras significativas, detección de errores de transcripción, verificación de consistencia interna, criterios de exclusión de datos (solo con evidencia documentada), y registro de decisiones sobre datos sospechosos.
