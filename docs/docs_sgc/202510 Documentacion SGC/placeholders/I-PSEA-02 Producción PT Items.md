## Descripción

Instructivo paso a paso para la producción/generación de los ítems de ensayo de aptitud. Para CALAIRE-EA, describe el proceso de generación de atmósferas controladas in situ mediante dilución dinámica a partir de CRM (cilindros de gas certificados). Incluye: configuración del sistema de dilución, cálculo de flujos, verificación de concentración objetivo, registro de condiciones ambientales y criterios de aceptación del ítem generado.
