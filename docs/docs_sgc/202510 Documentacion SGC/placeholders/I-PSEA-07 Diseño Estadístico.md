## Descripción

Instructivo técnico detallado que complementa P-PSEA-06 con los pasos operativos del diseño estadístico del esquema. Incluye: selección del algoritmo (A vs. B vs. C según n y distribución), cálculo de σ_pt usando la fórmula δ_E = max(a% × x_pt, b% × span), selección de scores (z, z', ζ), criterios de decisión para n variable (2–15+), y configuración del aplicativo R/Shiny para cada ronda.
