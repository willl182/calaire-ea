## Descripción

Formato para la confirmación formal de participación de un laboratorio en una ronda de EA. Incluye: datos del laboratorio participante, persona de contacto, analitos/niveles en los que participa, aceptación de términos y condiciones, declaración de confidencialidad y firma autorizada. Se envía al participante junto con las instrucciones pre-ronda (I-PSEA-09).
