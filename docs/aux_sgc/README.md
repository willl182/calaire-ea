# Documentación Consolidada SGC y Prueba Piloto

Este directorio es el punto único de consulta para la documentación del Sistema de Gestión de Calidad del esquema PSEA y para la documentación específica de las rondas de la prueba piloto de [[CALAIRE-EA]].

## 01_bloque_general

Documentación transversal del SGC. Aplica al esquema completo, no a una ronda específica.

| Carpeta | Contenido |
|---|---|
| `00_plantillas_base/` | Plantillas oficiales para documentos generales, procedimientos, instructivos y formatos. |
| `01_documentos_marco/` | Documento general del esquema y protocolo marco de participación. |
| `02_procedimientos/` | Procedimientos técnicos P-PSEA aplicables al esquema. |
| `03_instructivos/` | Instructivos técnicos generales del SGC. |
| `04_formatos_maestros/` | Formatos maestros F-PSEA no diligenciados por ronda. |
| `05_matrices_inventarios/` | Inventario, diccionario, árbol documental y matrices de cumplimiento normativo. |
| `06_procedimientos_gestion/` | Procedimientos de gobernanza del PEA: control documental, registros, riesgos, auditoría, revisión, quejas, apelaciones y proveedores. |

## 02_prueba_piloto_rondas

Documentación específica de la ejecución piloto. Aplica a la planificación, ejecución y cierre de rondas piloto.

| Carpeta | Contenido |
|---|---|
| `00_planificacion/` | Planes, cronogramas, formatos operativos maestros, instructivos y procedimientos usados para preparar las rondas piloto. |
| `01_ronda_simple/` | Copias diligenciables F-PSEA-05 a F-PSEA-14 para ronda simple CO/SO2 con P1. |
| `02_ronda_compleja_fase1/` | Copias diligenciables F-PSEA-05 a F-PSEA-14 para ronda compleja O3/NO/NO2 con P1 y P2. |
| `03_ronda_compleja_fase2/` | Copias diligenciables F-PSEA-05 a F-PSEA-14 para fase posterior CO/SO2 con P2. |
| `04_informes_operativos/` | Versiones del informe operativo de ejecución de la prueba piloto. |
| `05_evidencias_soporte/` | Diagramas, calendarios visuales, inducción SST y otros soportes operativos. |

## z_archivos_para_descartar

Zona de retención temporal para documentos que no aplican como documentación controlada ni como evidencia directa de la prueba piloto. No se eliminaron archivos; solo se aislaron para revisión posterior.

| Carpeta | Criterio |
|---|---|
| `01_borradores_ia/` | Propuestas, planes y borradores generados durante iteraciones anteriores. |
| `02_backups_duplicados/` | Copias de respaldo, duplicados y versiones intermedias. |
| `03_informes_antiguos/` | Informes de resultados históricos o versiones no vigentes. |
| `04_referencias_externas/` | Guías externas usadas como referencia, no documentos CALAIRE-EA. |
| `05_misc/` | Comunicaciones, listados, manuales o archivos sueltos pendientes de decisión documental. |

## Criterio de uso

- Para crear o actualizar documentación del SGC, partir de `01_bloque_general/00_plantillas_base/`.
- Para preparar una ronda piloto, usar `02_prueba_piloto_rondas/00_planificacion/` y copiar a la carpeta de la ronda correspondiente.
- Para auditoría documental, consultar primero `01_bloque_general/05_matrices_inventarios/`.
- Para depurar, revisar `z_archivos_para_descartar/` antes de eliminar cualquier archivo.
